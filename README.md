# CyAnime — Платформа для просмотра аниме

CyAnime — это современное веб-приложение для поиска и просмотра аниме, разработанное в рамках итогового проекта. Сервис объединяет интерфейс на основе Flexbox/Grid с функционалом облачной синхронизации и локального хранения данных.

## Основные функции
* Интерактивный поиск: Мгновенный поиск по базе аниме с использованием внешнего API.
* Система Избранное: Сохранение выбранных тайтлов в локальное хранилище браузера (LocalStorage).
* История просмотров: Синхронизация прогресса просмотра в реальном времени через Firebase Firestore.
* Авторизация: Личный кабинет пользователя с использованием Firebase Auth.
* Умный плеер: Автоматическое сохранение последней просмотренной серии и выбранной озвучки.
* Настройки интерфейса: Кастомизация внешнего вида приложения и параметров профиля.

## Технологии
* Frontend: HTML5 (семантическая разметка), CSS3 (Flexbox/Grid), JavaScript ES6+.
* Backend/DB: Firebase (Auth и Firestore) для облачного хранения данных пользователя.
* API: Интеграция с Kodik API для получения актуальной базы видеоконтента.
* Асинхронность: Использование async/await и fetch для обработки запросов без перезагрузки страниц.

## Структура проекта
Проект состоит из 7 функциональных страниц:
1. index.html — Главная страница с актуальными новинками.
2. search.html — Поиск и фильтрация контента по базе.
3. watch.html — Страница плеера с детальной информацией о тайтле.
4. favourite.html — Список закладок пользователя (реализовано через LocalStorage).
5. history.html — История просмотров с синхронизацией в облаке.
6. auth.html — Формы входа и регистрации пользователей.
7. settings.html — Настройки профиля и персонализация интерфейса.

## Инструкция по запуску
1. Для просмотра проекта в браузере перейдите по адресу: https://cyanime.site
2. Веб-приложение полностью оптимизировано для работы в современных браузерах (Chrome, Firefox, Safari).
3. Весь функционал, включая поиск и работу с API, доступен без дополнительной настройки со стороны пользователя.
